북한산 능선을 따라 가다보니 백운대, 인수봉을 새로운 각도에서 볼 수가 있어서 정말 좋았습니다. 그간 공부하느라 힘들었던 스트레스도 완전히 날려버렸습니다.



일단 시험방식이 크게 변경되었고, 만만해 보였던 문제들이 풀어보니 까다로운 함정들이 많이 숨겨져 있는데다, 시간까지 모잘라서 아주 고생했습니다.



처음보는 데이터분석 시험이라 몰랐는데, 데이터 시험은 문제보다는 어떤 데이터가 주어지는가가 더 결정적인 영향을 미치는 요소인 것 같습니다.



돌아보니, 데이터를 보기전에 문제가 쉬워보인다고 방심했던 게, 가장 후회가 됩니다.



특히 마지막 텍스트마이닝 부분은 제가 예상했던 부분이었는데도,

몇개는 시간 부족으로 못푼 것도 있어서, 너무 아쉽네요. ㅠㅠ



주최측에서 공식적으로 문제를 노출시키지 말아달라고 한데다가, 시험본 사람도 얼마안되어서, 시험관련 포스팅을 하는 것이 매우 “조심스럽습니다”.



그래서 일반적인 키워드 형태로만 정리를 했습니다.

어디를 중점적으로 공부해야 하는지 정도는 공유해도 될 것 같아서요.



1. 시험방식



1) 전체적인 실기 시험방식을 제1차에 비해서 완전히 바꾸어, 전체적인 분석을 통해서  도출된 시사점을 가지고, 최종 보고서를 내는 형식이 아니라,   data munging,  data  mining,         통계분석, text mining 등 4개의 주제별로 주어진 문제를 풀어서 답을 제출하는 형식으로 전환.



2) 4개 주제별로 각각 문제를 3~5개 정도 주고 각각의 답을 결과물 출력 혹은 코드로 제출함.



2. 시험문제 : 주최측에서 문제를 노출시키지 말아달라고 “공식적으로” 부탁을 해서,

문제를 posting하는 것은 아무래도 조심스럽네요. 하지만, 어디를 중점적으로 공부해야 하는지 정도는 공유해도 될 것 같아서, 조심스럽게 키워드 위주로만, 포스팅합니다.



1) data munging



주어진 데이터를 출제자가 요구한 형태의 출력이 나올 수 있도록 변형,  계산하여 정리한후, 집계 및 제출 (sqldf 패키지, plyr 패키지, reshape 패키지)



2) data mining



(1) 주어진 데이터를 가지고, 3개 이상의 예측모델링 기법을 사용하여 예측을 하고,

결과를 비교하여, 최적의 모델을 산정.  : decision tree(rpart 패키지), logistic regression(glm, family=”binomial”), randomForest(party 패키지)



(2) 주어진 데이터를 가지고, 그룹을 나눈 뒤, 각그룹을 연관분석으로 비교, 결과 해석 (arules 패키지)



3) 통계 분석 : 주어진 데이터를 가지고 변수선택 과정을 통해 최적의 regression 모델(lm)을 만들고, 도출된 모델의 파라미터들이 가지는 의미를 해석 (기본 stat 패키지)



4) text mining : 주어진 데이터를 가지고, 문제에서 지정한 문서를 골라내고, 이를 읽어들여, 명사를 추출하고, 제시된 방법으로 필터링한 후, 분석 결과를 그래프로 출력               (KoNLP패키지, tm 패키지)



3. 시험후 소감



1) 처음에는 너무 시험이 쉬워졌다는 생각이 들 정도로 문제가 만만해 보였으나, 실제로 풀어보니 곳곳에 함정이 있는 문제들이었음.



2) 데이터 양이 많은 것도 아니고, 그동안 안 다루어 봤던 것도 아니고, 실제로 예상하고 준비했던 문제 들도 많이 나왔는 데도 불구하고, 각각의 문제를 실제로 푸는 것 자체가 쉽지않은 일이었음.



3) 요구하는 분석기법의 갯수가 일단 많고, 각각의 기법의 장단점과 현실 적용시의 주의사항들을 잘 알고 있어야 문제를 풀 수 있는 형태로 까다롭게 출제를 하였음.



4) 거의 4시간 동안, PC가 다운되어 다른 PC로 바꾸는 동안 초코바 먹은 것 외에는 full로 정말 정신없이 했는데도, 문제를 다풀기 어려웠음.  문제가 많아지고, 답이 명확하게 나오는 형태로 문제가 출제되어 전체적으로 시간관리를 어떻게 하는 지가 중요한 이슈가 됨.



5) 데이터 분석전문가를 가려내는 것, 그 자체가 쉽지않은 일이라서, 시험방법은 앞으로도 변경의 여지가 남아 있는 것으로 보임.
